# Spot Trading

Learn how to trade cryptocurrencies on CoinByte's spot market.

## Understanding Order Types

### Market Order
- Executes immediately at best available price
- Guaranteed execution, but price may vary
- Best for quick trades

### Limit Order
- Set your desired price
- Executes only when price is reached
- Better price control, but no guaranteed execution

## How to Place a Trade

1. Go to the Trading page
2. Select your trading pair (e.g., BTC/USDT)
3. Choose order type (Market/Limit)
4. Enter amount to trade
5. Review order details
6. Confirm trade

## Trading Tips

1. Use limit orders for better prices
2. Monitor the order book
3. Check trading volume
4. Set stop-loss orders
