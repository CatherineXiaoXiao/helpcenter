# Price Alerts

Stay informed about market movements with CoinByte's price alerts.

## Setting Up Price Alerts

1. Go to the Markets tab
2. Select your desired cryptocurrency
3. Tap the bell icon
4. Choose alert type:
   - Price rises above
   - Price falls below
   - Percentage change
5. Set your target price
6. Enable notifications

## Types of Alerts

### Price Level Alerts
- Set specific price targets
- Receive notifications when reached
- Multiple alerts per asset

### Percentage Change Alerts
- Track price movements
- Set 1h, 24h, or 7d changes
- Customizable thresholds

## Managing Alerts

- View all active alerts
- Edit existing alerts
- Delete unnecessary alerts
- Temporary disable/enable

## Notification Settings

- Push notifications
- Email alerts
- SMS alerts (optional)
- Customize alert sounds

## Best Practices

1. Set realistic price targets
2. Don't set too many alerts
3. Regular review and update
4. Consider market volatility
