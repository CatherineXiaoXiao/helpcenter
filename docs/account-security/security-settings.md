# Account Security Settings

Learn how to configure comprehensive security settings for your CoinByte account.

## Account Security Dashboard

The Account Security Dashboard provides an overview of your account's security status. You can access it by:

1. Logging into your CoinByte account
2. Navigating to "Settings" > "Security"
3. Reviewing your current security score and recommendations

## Password Management

### Changing Your Password

1. Go to "Settings" > "Security" > "Password"
2. Enter your current password
3. Create a new password following the security requirements
4. Confirm your new password
5. Complete verification if prompted

### Password Security Requirements

- Minimum 12 characters
- Include upper and lowercase letters
- Include at least one number
- Include at least one special character
- Avoid using personal information or common words

## Login Protection

### Login Notifications

Enable notifications for:
- New device logins
- Unusual location logins
- Failed login attempts

### IP Whitelisting

1. Go to "Settings" > "Security" > "IP Whitelist"
2. Add trusted IP addresses
3. Enable IP restriction to only allow logins from whitelisted IPs

### Anti-Phishing Code

1. Go to "Settings" > "Security" > "Anti-Phishing"
2. Create a unique code
3. This code will appear in all official CoinByte emails

## Account Activity Monitoring

### Recent Activity Log

Review your recent account activity:
1. Go to "Settings" > "Security" > "Activity Log"
2. Check login history, including time, location, and device
3. Report any suspicious activity immediately

### Session Management

1. View all active sessions
2. Terminate any unrecognized sessions
3. Set automatic session timeout periods

## Security Notifications

Configure how you receive security alerts:
1. Go to "Settings" > "Notifications" > "Security Alerts"
2. Choose your preferred notification methods (email, SMS, app)
3. Select which types of events trigger notifications

## Frequently Asked Questions

### How often should I change my password?

We recommend changing your password at least every 3 months, or immediately if you suspect any security breach.

### What should I do if I notice suspicious activity?

If you notice any suspicious activity:
1. Change your password immediately
2. Enable or reset two-factor authentication
3. Contact our support team
4. Review and terminate all active sessions

### How can I maximize my account security?

For maximum security:
- Use all available security features
- Keep your contact information up to date
- Regularly review account activity
- Keep your devices and software updated
- Be vigilant against phishing attempts
