Add or remove a crypto withdrawal wallet address

All new crypto withdrawal addresses must be confirmed by email or Google Authenticator.

How to add a wallet address​

  • App
  • Web

App​

  1. Select the CoinByte icon in the top left to open the Personal center.
  2. Select your email address to open Account info.
  3. Select yManage withdrawal address.
  4. Select the currency, select Addn an address.
  5. Paste your withdrawal address or scan the code.
  6. Select Confirm; Then you need to do 2FA verification to finish.

Web​

  1. Select Asset in the header.
  2. Select Withdraw on the left-hand side menu.
  3. Select the cryptocurrency you want to withdraw.
  4. Select add an address to turn to Address Management page.
  5. Select Add a withdrawal address.
  6. Paste your withdrawal address from other platforms.
  7. Select Confirm, Then you need to do 2FA verification to finish.

How to remove a wallet address​

  • App
  • Web

App​

  1. Select Withdraw in the Dashboard.
  2. Select Crypto currency withdrawal, then select the cryptocurrency you want to withdraw.
  3. Select Please select an address to go to Address Management page.
  4. Swipe left the address you want to remove, then select "delete".

Web​

  1. Select Asset on the header.
  2. Select Withdraw on the left-hand side menu.
  3. Select the cryptocurrency you want to withdraw.
  4. Select add an address to go to Address Management page.
  5. Select delete for the address you want to remove.
